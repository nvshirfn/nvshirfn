## Featured Projects

### 🗣️ [Malay Meeting Summarizer](https://github.com/nvshirfn/meeting_summarizer_fyp)
NLP pipeline for Bahasa Malaysia meeting audio. Transcribes speech to text, normalizes colloquial Malay, then runs extractive and abstractive summarization alongside sentiment analysis and topic detection.

`Python` `Flask` `Malaya` `ELECTRA` `NLP`

---

### 👕 [SRD Detection System](https://github.com/nvshirfn/SRD-Detection-System)
Computer vision system that checks student compliance with UiTM's dress code (SRD). Detects collar, lanyard, and shoes in real-time via webcam or uploaded images using three fine-tuned YOLOv8 models.

`Python` `YOLOv8` `PyTorch` `OpenCV` `Flask`

---

### 🤖 [Cikgu Bot](https://github.com/nvshirfn/Cikgu-Bot) · [Legal Bot MY](https://github.com/nvshirfn/LegalBotMY)
AI-powered chatbot projects built using OpenAI GPT for educational (Year 6 Science) and legal Q&A contexts in Malaysia.

`Python` `Flask` `OpenAI API` `SQLite`

---

### 🎓 [UniEquip](https://github.com/nvshirfn/UniEquip-Web)

`PHP` `MySQL` `HTML` `CSS`
